<h1 align="center">AutoGnirehtet - Automatic reconnect script for gnirehtet</h1>

## Building
This project is configured for setuptools

## Usage :

Use
```shell_session
$ auto-gnirehtet
```
instead of
```
$ gnirehtet run
```
